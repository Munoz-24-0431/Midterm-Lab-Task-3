Midterm Lab Task 3

Interactive Data Science Salary & Job Trends Dashboard

Project Overview

This project focuses on creating an interactive Data Science Salary & Job Trends Dashboard using Excel. The dashboard analyzes job distribution, salary trends by role and company size, and sector-wise employment patterns through pivot tables, charts, slicers, and map visualization.

Step 1 - Data Preparation

Use the Given Data

  • Open the Worksheet on Uncleaned DS Jobs
  • Access the Transform Tables: Sal By Role
  • Access the Transform Tables: Sal By State
  • Access the Transform Tables: Sal By Size

Step 2 - Dashboard Design Requirements

Key Metrics to Highlight

  • State with the most Data Science jobs
  • Job role with the highest average salary
  • Company size that pays the highest salary
  • Sector with the lowest & highest number of Data Science jobs
  • Sector with the minimum & maximum average salary

Step 3 - Pivot Table Creation

  • Create pivot tables to summarize sector jobs data
  • Create pivot tables to analyze salary distributions
  • Set up proper aggregation methods for accurate analysis

Step 4 - Chart Generation

  • Create bar charts for salary comparisons
  • Use column charts for job distribution
  • Implement pie charts for sector analysis

Step 5 - Interactive Elements

  • Add slicer for Role Type filtering
  • Add slicer for Company Size filtering
  • Add slicer for State filtering
  • Ensure proper connections between slicers and visualizations
